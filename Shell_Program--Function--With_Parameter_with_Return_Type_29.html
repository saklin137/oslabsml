Function With Return type and With Parameter

Note : There is no return type in shell like C, java, etc but we can achieve return via below syntax

functionName

{

var=$1

var=$2

stm2

stm3

......

.....

}

variableName=$(functionName parameter1 parameter2)

eg:

add()

{

a=$1

b=$2

sum=$(($a+$b))

echo “$sum”

}

result=$(add 5 6)

echo “$result”
