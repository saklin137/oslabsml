library


  • In C programming fork() function is defined under the library “unistd.h”
  • fork() function only work for linux environment.
  • For knowing the process id we use getpid() function in C. getpid() funtion is defined under “unistd.h”
  • getpid() function return integer.
  • getpid() function return process id which is integer.

•Below code is an example of how to create a child process.

#include<stdio.h>
#include<unistd.h>
int main()
{
int childid = fork();
if(childid==0){
printf("I am child Process, my id is %d,and my child is is %d\n",getpid(),childid);
}else{
printf("I am Parent Process, my id is %d,and my child is is %d\n",getpid(),childid);
}
return 0;
}
