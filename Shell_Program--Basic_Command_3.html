How To Execute your Shell Program

eg:

./hello.sh

way2 -> sh filename.fileextension

eg:

sh hello.sh

cp -> copy

it is use for copy the folder/file

cp <filename/folder name> <newfilename/folder name>

eg : cp new.sh test.sh

cp filename destination

eg : cp test.sh /home/student/os

mv -> move

it is use for renaming and moving the folder/file

mv <filename/folder name> <newfilename/folder name>

eg : mv new.sh test.sh

mv filename destination

eg : mv test.sh /home/student/os

How to install any package

sudo -> superUser Do

apt -> application package tool

way1:

sudo apt install <package-name>

eg:

sudo apt install gcc

sudo apt install openjdk-8-jdk

sudo apt install vlc

way2:

After download debian package file from the internet,

open terminal at that folder where your downloaded package is downloaded

and then run below command for install your package

sudo dpkg -i hello.deb

way3:

using snap

snap - single node application package

eg:

sudo snap install chromium

How to uninstall any package

way1:

sudo apt remove/purge <package-name>

eg:

sudo apt remove gcc

sudo apt purge gcc

sudo apt autoremove

way2:

sudo dpkg -r <package-name>

eg:

sudo dpkg -r hello.deb

way3:

sudo snap remove <package-name>

eg:

sudo snap remove spotify

cat -> concatenate

How to show file content using cat?

syntax : cat <filename.fileextension>

eg:

cat hello.txt

cat new.sh

How to create file using cat Command?

syntax : cat > filename.fileextension

> it is use for write out any file and if same name file already present in the folder. then its fully remove existing content

eg:

cat > swap.sh

a=15

b=36

a=$(($a+$b))

b=$(($a-$b))

a=$(($a-$b))

echo “a=$a b=$b”

for save press Ctrl + D

How to concatenate one file content to another file?

let in folder i have two file

one file name is new.txt

another file name is final.txt

and i want to append new.txt all content to final.txt

then run the below command

>> its use for write out the file without removing the existing content

cat new.txt >> final.txt

tail -> use for showing the last 10 line of the content

syntax: tail filename.fileextension

syntax : head filename.fileextension

it by default show first 10 line of your file content

eg:

head hello.txt

eg:

| -> this symbol use for combining more than one command

sort

syntax : sort filename.fileextension

it sort all line of the file content by alphabetically order in ascending order

eg: sort new.txt

syntax : sort -r filename.fileextension

it sort all line of the file content by alphabetically order in descending order

eg: sort -r new.txt
