Vi Editor

Insert Mode Commands

i -> Insert before cursor

I -> Insert at beginning of line

A -> Append at end of line

O -> Open new line above

Esc -> Exit Insert Mode

Navigation Commands

h -> move left

l -> move right

j -> move down

k -> move up

0 -> start of line

$ -> end of line

w -> next word

b -> previous word

G -> go to end

gg -> go to top

Editing Text

x -> delete character

dd -> delete line

yy -> copy line

y -> copy

v -> visual mode

p -> paste

u -> undo

Ctrl + r -> redo

Save and Quit

:w -> Save

:q -> Quit

:wq -> Save & Quit

:q! -> Quit without save

:x -> Save and Exit
