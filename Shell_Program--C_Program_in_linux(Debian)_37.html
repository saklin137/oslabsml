C Program in linux(Debian)


How to Run C program in linux(debian)

  1. Open terminal and make file using touch

touch welcome.c

  1. open program using editor like gedit, nano

nano welcome.c

3)In editor write your c program and save it 

#include<stdio.h>
int main()
{
printf("Welcome buddy");
return 0;
}



  1. For compiling C program use gcc
      1. check gcc is present or not
        1. if not present install it using “sudo apt install gcc”
        2. either compile your program using gcc
  2. for compile your program type gcc filename.c

eg: gcc welcome.c

it generates one executable file name as “a.out”

  1. for run your program type ./executable_file name

eg ./a.out
